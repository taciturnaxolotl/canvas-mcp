Canvas MCP Server

Connect your Canvas LMS to AI assistants via the Model Context Protocol

Features

  • Direct integration with Claude Desktop and other MCP clients
  • Personal Access Token authentication (no admin access needed)
  • Access courses, assignments, and grades from your AI assistant
  • Works with any Canvas institution

Get Started

How to get a Personal Access Token
  1. Log in to your Canvas account
  2. Go to Account → Settings
  3. Scroll to "Approved Integrations"
  4. Click "+ New Access Token"
  5. Set Purpose: "MCP Server"
  6. Set Expires: (optional, max 120 days)
  7. Click "Generate Token"
  8. Copy the token and paste it above
